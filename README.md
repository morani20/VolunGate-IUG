# 🎓 IUG VolunGate Platform

A modern web application designed to manage and organize volunteer activities at the **Islamic University of Gaza (IUG)**.

---

## 🚀 Project Overview

The **VolunGate System** is a platform that helps universities efficiently manage volunteer work by providing:

* 👥 Volunteer management
* 📋 Task assignment and tracking
* 📊 Reports and performance monitoring
* 🧩 Role-based system (Admin / Supervisor / Student)

---

## 🛠️ Tech Stack

* ⚛️ React + Vite
* 🧠 TypeScript
* 🎨 CSS (Custom UI + Animations)
* 🔀 Wouter (Routing)

---

## ✨ Key Features

* 🔐 Role-based access system
* 📊 Dashboard for statistics
* 📋 Task management system
* 📑 Reports and analytics
* 🔔 Interactive UI and feedback system
* 📱 Fully responsive design
